# 武汉大学机器人学院学科资料分享计划 (WHU Robotics Course Materials)

欢迎来到 **武汉大学机器人学院学科资料分享计划**！本项目旨在收集、整理并分享武汉大学机器人学院及相关学科的优质课程资料、课件 PPT、历年真题及答案，为广大同学提供一份系统、完备的学习资源指南。

## 📁 资料分类与目录结构

目前仓库已归档以下课程的相关资料，您可以点击对应的文件夹进行查看和下载：

### 1. 📘 计算机编程类
*   **[C语言](./C语言)**
    *   汇集了历年（10-25学年）C语言（及计算机原理与编程基础、测绘程序设计等）的期中、期末模拟试卷及参考答案。
    *   包含 C 语言必背经典程序、各章节复习题、程序设计上机题库等核心复习资料。
*   **[C语言内容课件](./C语言内容课件)**
    *   包含30余讲详实的 C 语言精讲课件（PDF版）。
    *   涵盖分支循环、函数递归、指针深入剖析、结构体/联合体、动态内存管理、单双向链表，以及扫雷、贪吃蛇等项目实践。

### 2. 🧮 数学基础课
*   **[高数A上](./高数A上)**
    *   **历年真题**：收录了 20-25 学年近五年的期中考试真题试卷及详细解析。
    *   **课件与视频**：涵盖不定积分、定积分、反常积分等重点章节的 PPT 课件及部分知识点讲解视频（如阿基米德）。
*   **[线代A](./线代A)**
    *   包含线性代数 A 的相关课件、复习资料与试题集。

### 3. ⚙️ 工程专业基础
*   **[《机械工程图学习题集》答案](./《机械工程图学习题集》答案)**
    *   整理了《机械工程图学习题集》各核心章节（第1、4、5、8、9章）的详细答案。
    *   包含手绘作业参考、剖视图解析以及装配图（如减速器装配图）等高难资料。

## 🌟 参与贡献 (Contributing)

开源与分享让知识得以传承。我们非常期待你的加入！如果你有其他课程的优秀笔记、历年试卷或课件，欢迎提交：

1. **Fork** 本仓库。
2. 将您拥有的资料放入对应课程的文件夹下（如果是新课程，请新建分类文件夹）。
3. 提交一个 **Pull Request** 并附上简单的说明。
4. 我们会在审阅后合并您的分享，让更多学子受益。

## ⚠️ 免责声明

*   本仓库内所有的试卷、课件及习题答案仅供**内部学习与交流使用**，请勿将资料用于任何商业盈利目的。
*   历年试题回忆版或个人编写的答案仅供参考，不保证 100% 正确，请务必结合教材和授课教师的指导进行备考。
*   若发现资料中存在错漏，欢迎通过 Issue 或 PR 进行指正。
